## Misc

On OSX, the following may be necessary to make sure that CMake finds the versions of bison and flex installed with homebrew.

```bash
export CMAKE_PREFIX_PATH=`brew --prefix`/opt/bison/bin:`brew --prefix`/opt/flex/bin
```
